# Landing Page | Lutiane Flores

Landing page comercial criada a partir do PDF `Midiakit Lutiane.pdf`, preservando a direção visual editorial do material original e reorganizando o conteúdo para conversão.

## Como abrir localmente

1. No terminal, entre na pasta do projeto:
   `cd /Users/rodrigoribeiro/Luti`
2. Rode um servidor estático:
   `python3 -m http.server 4173`
3. Abra:
   `http://localhost:4173`

## Estrutura

- `index.html`: conteúdo da landing
- `styles.css`: visual, responsividade, hover e motion visual
- `script.js`: reveal on scroll, counters e tilt interaction
- `assets/extracted/`: imagens extraídas do PDF original

## Observação

As fontes usadas no PDF foram replicadas com `Yeseva One` e `Montserrat`.
